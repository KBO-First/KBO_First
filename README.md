# ⚾ KBO First — 당신의 첫 야구 직관 가이드

> 룰부터 좌석까지, 응원봉부터 티켓팅까지 — 야구 입문자를 위한 모든 것

웹 클라이언트 컴퓨팅 3조 | 2026

---

## 📌 프로젝트 소개

야구를 즐기고 싶지만 어디서부터 시작해야 할지 막막한 20대 입문자를 위한 야구 직관 가이드 웹사이트입니다.

**타겟 사용자**
- 야구 지식이 없는 20대 입문자
- SNS 인증샷을 찍고 싶지만 '가짜 팬'처럼 보일까 고민하는 대학생

**페르소나** — 김민지 · 23세 · 대학생
- 친구들의 야구 유니폼 사진을 보고 가보고 싶지만 야구 지식이 없음
- 내 성향에 맞는 구단을 추천받고 싶음
- 실패 없는 직관을 위한 준비물과 트렌드를 파악하고 싶음
- 가장 큰 장벽인 '티켓팅'을 미리 연습해보고 싶음

---

## 🗂 페이지 구성

### 🏠 메인 페이지 `main.html`
- KBO 흥행 데이터 & 팬덤 순위 TOP 5
- 최신 야구 응원 트렌드 3선
- 직관 필수 준비물 체크리스트 6개
- 날씨 · 우천 TIP 카드

### 👥 팀원 소개 `team.html`
- 야구 포지션 비유 팀원 카드 5장
- 카드 flip → 역할에 맞는 한 마디
- 팀 컬러 그라디언트 배경

### 🔮 팀 추천 폼 `form.html`
- 10가지 O/X 성향 질문
- 진행률 바 + 도트 인디케이터
- 4가지 성향 유형 구단 매칭
- 결과 공유 기능

### 🎮 티켓팅 게임 `game.html`
- 60초 제한 타이머
- 야구장 좌석 맵 CSS Grid
- 깜빡이는 타겟 좌석 클릭
- 미스 3회 → GAME OVER

## 🔗 페이지 간 연결 구조

**주제 ↔ 폼 — 진입 장벽 해소**
야구 입문의 첫 장벽인 '나의 팀' 정하기를 라이프스타일 기반 O/X 폼으로 해결해 소속감을 만든다.

**주제 ↔ 게임 — 실전 역량 강화**
팀을 정한 뉴비가 마주치는 두 번째 장벽 '티켓팅'을 긴장감 넘치는 시뮬레이션으로 심리적 장벽을 낮춘다.

**폼 ↔ 게임 — 개연성 완성**
폼에서 '목적지(팀)'를 찾고, 게임에서 '티켓'을 거머쥐는 행동으로 이어져 팬심이 실제 직관 의지로 강화된다.
